# Desafio Herói - DIO

## Objetivo

Criar uma classe genérica que represente um herói de aventura, com as seguintes propriedades:

- **nome**  
- **tipo** (ex: guerreiro, mago, monge, ninja)

Além disso, deve ter um método chamado **atacar**, que exiba a mensagem:
{o tipo} atacou usando {ataque} 


O ataque deve variar conforme o tipo:

- **Mago** → usou magia  
- **Guerreiro** → usou espada  
- **Monge** → usou artes marciais  
- **Ninja** → usou shuriken

---

## Tecnologias utilizadas

- JavaScript  
- Node.js (para executar o código)  
- Git e GitHub (para versionamento e entrega)  

---

## Como executar

1. Certifique-se de ter o **Node.js** instalado:  
```bash
node -v
2. Abra a pasta do projeto e execute:
node index.js

3. O resultado esperado será:
Mago atacou usando magia
Guerreiro atacou usando espada
Monge atacou usando artes marciais
Ninja atacou usando shuriken
